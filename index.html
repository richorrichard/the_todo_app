*Classic To Do List

+ #

POINT

>CLICK< the plus sign to add a task.
>CLICK< the hash to clear completed tasks.
>CLICK< to edit task.
>CLICK< the dot to mark task complete.

TYPE

ENTER to create new task from within task.
TAB to make current task a child of the task above.
SHIFT + TAB to un-nest current task one level.
CMD + ENTER to mark current task complete.
CMD + DELETE to delete current task.
CMD + ↑ to navigate to next task.
CMD + ↓ to navigate to previous task.
